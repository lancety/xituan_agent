# FontAwesome 图标转 PNG 工具

这是一个将 FontAwesome Solid 图标转换为指定颜色 PNG 图片的在线工具。

## 功能特点

- 🎨 **颜色自定义**：支持任意颜色选择
- 📏 **多尺寸导出**：支持 64×64、81×81、128×128、256×256 像素
- 🔍 **图标搜索**：快速搜索和选择图标
- 📱 **微信优化**：81×81 像素专为微信小程序 tabBar 优化
- 💾 **批量导出**：一键导出所有尺寸

## 使用方法

1. 打开 `fontawesome-icon-converter.html` 文件
2. 在搜索框中输入图标名称（如：home, cart, user）
3. 点击选择想要的图标
4. 调整图标颜色
5. 选择导出尺寸（推荐 81×81 用于微信小程序）
6. 点击"导出 PNG"按钮

## 导出文件

导出的 PNG 文件会自动下载到浏览器默认下载目录，文件名格式为：
- `fa-{图标名}-{尺寸}x{尺寸}.png`

例如：`fa-home-81x81.png`

## 微信小程序使用

导出的 81×81 像素图标可以直接用于微信小程序的 tabBar 配置：

```json
{
  "tabBar": {
    "list": [
      {
        "pagePath": "pages/index/index",
        "text": "首页",
        "selectedIconPath": "images/fa-home-81x81.png"
      }
    ]
  }
}
```

## 注意事项

- 工具使用 FontAwesome 6 字体
- 导出的图标为纯色图标，适合用作 tabBar 图标
- 建议使用对比度较高的颜色以确保在不同背景下清晰可见
- 微信小程序 tabBar 推荐使用 81×81 像素的图标
